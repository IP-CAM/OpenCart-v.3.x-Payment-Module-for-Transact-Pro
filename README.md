# OpenCart 3.x payment module for Transact Pro

This module is using to connect OpenCart to Transact Pro Gateway payment system.

### Requirements

- This module depends from Transact Pro Gateway v3 PHP client library [GitHub](https://github.com/TransactPRO/gw3-php-client)

## Installation

Install module using standard OpenCart module installer or by manual way:
unzip archive to your OpenCart installation directory (no files will be overwriten).

## Basic usage

Enable and configure module in OpenCart admin (Extensions > Paymens > Transact Pro)
